# atcoderを解くときのディレクトリ作成ツール

## 概要
atcoderの解く番号のディレクトリが作成され、そのディレクトリに`modto.py`をコピーしたPythonファイル`a.py` ~ `f.py`を自動で作成する。  
`modto.py`を自分が好きなように編集してください。

## デモ
以下のようにコマンドをうつと、以下のようなディレクトリが自動で作成されます。
```bash
$ python make.py
atcoder何番のフォルダを作成する？123
ディレクトリ123を作成しました。    
```
<pre>
├─123
│      a.py
│      b.py
│      c.py
│      d.py
│      e.py
│      f.py
</pre>
